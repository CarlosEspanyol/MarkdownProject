## 1. Estilo de Código

- Usa **4 espacios** para la indentación.
- Nombres de variables y funciones en **camelCase**.
- Comentarios claros y concisos.
- Mantén las líneas de código con un máximo de **80 caracteres**.
- Formato de comillas: usa **dobles** para cadenas (`"texto"`).

Ejemplo:

```python
def sumarNumeros(a, b):
    """Suma dos números y devuelve el resultado."""
    return a + b
